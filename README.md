# Patient Portal Documentation Sample

This repository contains a short documentation sample created to demonstrate structured documentation for digital health tools. The content shows how user focused instructions can help new patients understand key portal features.

## About this sample
Purpose: provide concise, user friendly content for patient facing tools.  
Audience: patients and non technical users who need clear instructions.

## Files
- README.md (this file)
- patient-portal-overview.pdf (one page documentation sample)
- patient-portal-quick-guide.html (HTML documentation sample shown in portfolio)
- screenshot-html-page.png (optional screenshot used for the assignment)

## How to use
1. Review the PDF to understand the feature overview.  
2. Open the HTML file for an example of clear step by step documentation structure.  
3. Use the assets as reference points for writing simple, accessible user instructions.
